Web Design Steps

  • Goal identification:
    Determine what goals the site needs to fulfil. That is, what is the purpose of the site?
  • Scope definition:
    Once we know the site's goals, we can define the scope of the project. What pages and features the site requires to fulfil the goal, and the timeline for building these pages.
  • Sitemap and wireframe creation:
    Define the sitemap, specifying how the content and features defined in the scope definition will interrelate.
  • Content creation:
    Start creating content for the individual pages, always keeping search engine optimization in mind to help keep pages focused on a single topic.
  • Visual elements:
    Start working on the visual brand. Depending on the client, this may already be well-defined, but you might also be defining the visual style from scratch. Tools like style tiles, moodboards, and element collages can help with this process.
  • Testing:
    Time to make sure it all works. Combine manual browsing of the site on a variety of devices with automated site crawlers to identify everything from user experience issues to simple broken links.
  • Launch!
    Time to plan and execute your site launch! This should include planning both launch timing and communication strategies.
